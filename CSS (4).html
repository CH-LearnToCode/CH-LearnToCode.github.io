CSS Padding & Margins


Padding is used to create blank space inside a element
while a Margin is used to create blank space outside the borders of the element

Padding


The padding property specifies the amount of empty space in an element

The padding properties of an element can be specifyed with
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Margins


The margin property specifies the amount of white space around an element.

The margin properties of an element can be specifyed with
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
