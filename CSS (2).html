CSS Borders


This border property allows you to specify style, width and color of an elements border You can specify which side is affected my the border property with border-top for the top side, -bottom for bottom, -left and -right for left and right.
Or you can do border-(property): [top bottom value] [left right value] Or border-(property): [top] [right] [bottom] [left]

Border Styles


The style of the border can be specified with border-style

  • dotted - Defines a dotted border
  • dashed - Defines a dashed border
  • solid - Defines a solid border
  • double - Defines a double border
  • groove - Defines a 3D grooved border. The effect depends on the border-color value
  • ridge - Defines a 3D ridged border. The effect depends on the border-color value
  • inset - Defines a 3D inset border. The effect depends on the border-color value
  • outset - Defines a 3D outset border. The effect depends on the border-color value
  • none - Defines no border
  • hidden - Defines a hidden border


Width and Color


border-width: Defines the width of the border
border-color: Defines the color of the border

Rounded Borders


The border-radius is used to round out borders, the higher the value the more rounded.

All Border Properties
