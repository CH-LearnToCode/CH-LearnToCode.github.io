CSS Overflow

The overflow property defines whether the content inside a container will be cut if it's to big to fit in the container.

The overflow property has the following values:
  • visible - defualt, the content will not be cut
  • hidden - the content will be cut
  • scroll - the content is cut but is viewable via the scrolling
  • auto - same as scroll except only when necerssery
The overflow propery only works on block elements with a specified height.




overflow: visible

When overflow is set to visible the content that overflows out of the container will be shown.

overflow: hidden

The overflow property is set to hidden and so all content that overflows will be cut.

overflow: scroll

The overflow property is set the scroll and now it will hide the overflowing content but it can still be viewed if you scroll down.

overflow: auto

The overflow is set to auto, this means that it will display as scroll but only if neccersery so if it doesn't over flow it won't have the scroll bars



We can also use overflow-x and overflow-y individualy if we wanted to do so.
