Comparison Operations

OperatorExampleWhat is does
< x < y if x is smaller than y
> x < y if x is bigger than y
<= x < y if x is smaller or equal to y
>= x < y if x is bigger or equal to y
== x < y if x is equal to y (regardless of type, so 1 is equal to '1'
=== x < y if x is equal to y (type matters, so 1 is not equal to '1'
! !(x == y) if x is not equal to y
|| (x == y || x == z) if x is equal to y or z
&& (x == y && x == z) if x is equal to y and z
